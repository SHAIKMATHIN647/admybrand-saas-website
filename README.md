## ✨ Features

- 🔥 Hero Section with headline, subtext, CTA, and hero image
- 💡 Features Section with 6+ icons and descriptions
- 💳 Responsive Pricing Cards (3 tiers)
- 🌟 Testimonials Carousel
- ❓ FAQ Section with collapsible toggles
- 📞 Contact Form with validation
- 📱 Fully responsive and mobile-first
- 🎨 Glassmorphism + modern UI animations
- ⚙️ Reusable Component System (8+ components)

---

## 🛠️ Tech Stack

- **Next.js 14+ with App Router**
- **TypeScript**
- **Tailwind CSS**
- **Reusable Components**
- **Framer Motion** (for animations)
- **Responsive design** using Mobile-First approach

```bash
# Clone the repository

# Install dependencies
npm install

# Run development server
